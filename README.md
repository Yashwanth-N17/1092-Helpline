# 1092 AI Helpline (Modernized)

A high-performance, AI-powered emergency response system that bridges the gap between citizens and responders using WebRTC and an advanced AI Intelligence Layer.

## Project Structure

- **frontend**: React + Vite + Tailwind CSS dashboard and SOS interface.
- **backend**: Node.js modular server (Routes-Controllers-Services) handling low-latency AI Reply generation (Gemini 2.5 Flash).
- **ai_service**: Python FastAPI service for heavy analysis using DeBERTa v3 and DeepSeek Reasoner.

## Intelligence Layer Architecture

1. **Backend (Node.js)**:
   - **Gemini 2.5 Flash**: Real-time AI response generation for low latency.
   - Orchestrates communication between frontend and AI Service.
2. **AI Service (FastAPI)**:
   - **DeBERTa v3 Zeroshot**: AI Analysis & Severity Detection (LOW-CRITICAL).
   - **DeepSeek Reasoner**: Detailed Case Summary Generation.

## Getting Started

### Prerequisites
- Node.js (v18+)
- Python (3.9+)
- Gemini & DeepSeek API Keys

### Installation & Run

#### 1. AI Service (Port 8001)
```bash
cd ai_service
pip install -r requirements.txt
uvicorn app.main:app --port 8001 --reload
```

#### 2. Backend (Port 8000)
```bash
cd backend
npm install
npm run dev
```

#### 3. Frontend (Port 5173)
```bash
cd frontend
npm install
npm run dev
```

## Features
- **SOS Page**: Public-facing WebRTC emergency link.
- **Agent Dashboard**: Real-time active call tracking with AI insights.
- **Severity Detection**: Automatic classification of urgency level.
- **Case Summarization**: Intelligent summaries generated by DeepSeek.
- **Interactive AI Reply**: Instant feedback to citizens using Gemini.
