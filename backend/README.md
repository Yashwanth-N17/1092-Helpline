# 1092 Helpline - Backend Service

This is the primary backend for the 1092 Helpline, built with **Node.js**. It follows a modular architecture and coordinates between the frontend and the specialized AI Service layer.

## Features

- **AI Reply Generation**: Powered by **Gemini 2.5 Flash** for low-latency responses.
- **AI Service Integration**: Proxies requests to the FastAPI `ai_service` for heavy processing.
- **Modular Architecture**: Built with Routes, Controllers, and Services.

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **AI SDK**: @google/generative-ai
- **HTTP Client**: Axios

## Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation
1. Navigate to the directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   ```bash
   cp .env.example .env
   ```
   *Edit `.env` with your API keys and the correct `AI_SERVICE_URL`.*

### Running the Service
```bash
npm run dev
```
The server will run on `http://localhost:8000`.

## API Request Examples

### 1. Generate AI Reply
**Endpoint**: `POST /api/v1/replies/generate`
**Request**:
```json
{
  "context": "Citizen is reporting a theft.",
  "user_message": "My phone was snatched near the station."
}
```
**Response**:
```json
{
  "reply": "I have noted the incident. Please stay at the location, a patrol unit is being dispatched."
}
```

### 2. Analyze Case (Proxies to AI Service)
**Endpoint**: `POST /api/v1/cases/analyze`
**Request**:
```json
{
  "text": "The tree has fallen on the road."
}
```
**Response**:
```json
{
  "top_label": "Infrastructure",
  "severity": "MEDIUM",
  "intent": "Road blockage",
  "confidence": 0.88
}
```

### 3. Summarize Case (Proxies to AI Service)
**Endpoint**: `POST /api/v1/cases/summarize`
**Request**:
```json
{
  "transcript": "Full call transcript text..."
}
```
**Response**:
```json
{
  "summary": "Concise summary generated by DeepSeek."
}
```
